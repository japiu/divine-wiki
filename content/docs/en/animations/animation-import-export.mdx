#  Importing and Exporting Animations

This guide will help you import animations in Maya and export them for League modding.

| Name | Description |
| :--- | :--- |
| **RitoBin** | Converts .bin files to .py and back using ritobin_cli. |
| **Autodesk Maya** | Industry-standard 3D software, used for rigging and UV mapping. Perfect for League Modding. |

---

### Before you begin animation stuff

> **Note:** I strongly suggest you first get familiar with how to import .skn and .skl.

##  Importing

### Extracting what you need

Firstly, you want to extract your desired Champion folder with Obsidian (for this post, I will be working with Blitzcrank).

<img src="/wiki-images/1767108155362-3n6wzr.png" alt="Image" width={500} />

Once extracted, navigate to the location where you extracted it and find the animation folder (Should be something like `assets\characters\character\skins\base\animations`).

###  Importing the animation in Maya

Make sure these are your import options:

<img src="/wiki-images/1767108174480-9ghj4eg.png" alt="Image" width={500} />

With the model in the scene (import the .skn and .skl first), simply drag and drop the animation you want (should be `.anm`) to see in the viewport, and **VOILA**, you have your funny Blitzcrank walk in the Maya scene.

<img src="/wiki-images/1767108195951-qwxim.gif" alt="Image" width={500} />

##  Exporting

###  Setting up the timeline

Now, if you want to Export an animation, you will want to specify the frames you want to export. To do that, select the range you want to export in the timeline.

<img src="/wiki-images/1767108208069-z1qffc.png" alt="Image" width={500} />

Here, I made a small animation to demonstrate. The animation lasts for 20 frames, but the frames are between frame 40 and 60. To select the frames you want to export, make sure the two numbers are the same in both boxes, so I type 40/40 and 60/60 (this will change depending on YOUR animation length).

<img src="/wiki-images/1767108223105-k5cqfc.gif" alt="Image" width={500} />

**This is not what you want:**

<img src="/wiki-images/1767108229276-5l6rr.png" alt="Image" width={900} />

**This is what you want:**

<img src="/wiki-images/1767108233963-87tpbv.png" alt="Image" width={900} />

> The bar needs to be "full"

You also need to leave one empty frame at the start! So in my case, instead of 40/40, I would have 39/39.

### Exporting the animation

**Make sure you export in 30 fps!**

To export, select **File > Export All**.

<img src="/wiki-images/1767108240116-sli2y6.png" alt="Image" width={500} />

Then, select the animation you want to replace and make sure you export as `.anm`.

<img src="/wiki-images/1767108249814-9jfts4.png" alt="Image" width={900} />

Annnnnd well done! You now know how to import and export animations for League models!
