**Stitch** is a feature inside Hacksaw that allows you to borrow particle systems from one `.bin` file and insert them into another. This makes it possible to mix and match effects between champions, skins, and abilities — without having to build particles from scratch!

**[Quartz](https://github.com/RitoShark/Quartz)** is recommended, since it ports texture files automatically.

## What Stitch Allows You to Do

- Import existing particle systems from any champion or skin
- Build new visual effects by combining different sources
- Speed up modding without manually editing particle logic
- Create new thematic skins by blending multiple VFX sets
