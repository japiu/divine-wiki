**Blender Version:** 4.5.1  
If you are using a newer version, some steps may look slightly different.

This guide walks you through preparing a custom 3D model in Blender and replacing a League of Legends character model while keeping the original rig and animations.

## Requirements

| Tool                        | Purpose                                                                          |
| --------------------------- | -------------------------------------------------------------------------------- |
| **Lol Aventurine Plugin**   | Required to import/export `.skn` and `.skl` files in Blender                     |
| **Blender**                 | Main 3D software for editing the champion model                                  |
| **CATS Blender plugin**     | Not required, but useful for transferring weights between bones and other things |
| **Paint.net / Photoshop**   | Edit and export textures (`.dds`)                                                |
| **Paint.net `.tex` plugin** | Needed to open and edit `.tex` files                                             |
| **LtMAO**                   | Convert League `.tex` files to `.dds` and back, alternative to the plugin above  |
| **Flint**                   | Extract original League champion files (`.wad.client`)                           |
| **Jade**                    | Edit `.bin` files if skin adjustments are needed                                 |
| **Celestial**               | Load and test the custom skin in-game                                            |

**FAQ:**

I'm trying to do X why it not working please help?

- The answer for why is already in the guide 90% of the time. Re-read carefully and if its still unclear DM and ask.

Why can't I do X in edit mode?

- You had likely selected the wrong object. If you are editing the mesh make sure you select the mesh and not, for example, the armature before going into edit mode.

Do I select the green armature icon or the orange one in the objects list top right?

- Orange; The green icon can be ignored.

## 0. Prepare Your Model

### Combine Meshes

First import the model you want to swap to. If your model consists of multiple objects:

- Select all objects:
  <img src="/wiki-images/1768738897687-iknswq.png" alt="Image" width={200} />
- Press **Ctrl + J** to combine them into a single mesh

---

### Texture Display & Materials

- Enable material preview (viewport shading)
  <img src="/wiki-images/1768738971018-am18ih.png" alt="Image" width={200} />
- Select the model and open the **Materials** tab:
  <img src="/wiki-images/1774321805697-zoe5nl.png" alt="Image" width={100} />
- For models with multiple materials (if the materials do not have the textures assigned already, so like if the model is say just solid color):
  - Assign the correct texture to each material slot:
    <img src="/wiki-images/1768739020944-23h6dv.png" alt="Image" width={200} />
- If you plan to bake lighting or shadows into the texture, do it at this stage; I really recommend it if you are going for a "realistic" look.
  (No Divine guide for it yet, but tutorials are out there)

---

### Face Orientation

- Enable **Face Orientation** in the viewport overlays:
  <img src="/wiki-images/1768739336655-6mggq.png" alt="Image" width={200} />
  If the whole model turns red = correct normals orientation, you can skip this
  step.

If parts don't change color:

1. Go to edit mode
   <img src="/wiki-images/1768739510869-uxmh9.png" alt="Image" width={200} />
2. Enable **X-Ray**
   <img src="/wiki-images/1768739569367-b9oiaw.png" alt="Image" width={200} />
3. Select the full mesh (drag select)
   <img src="/wiki-images/1768739558165-jpsxo.png" alt="Image" width={200} />
4. Press **Alt + N → Flip**

If only some areas are not red, select those faces individually and flip them.

Disable Face Orientation once done.

---

### Merge Vertices (IMPORTANT)

- Enter **Edit Mode**
- Select the full mesh
- Press **M → By Distance**
- In the window that pops up bottom left, set the distance to **0.001 – 0.005**
  <img src="/wiki-images/1768739708348-3wx76a.png" alt="Image" width={200} />

This step removes duplicate vertices and prevents major issues later.

---

### Decimate (Vertex Limit)

League models must be under ~65,000 vertices to be imported.

1. Enable **Statistics** in the viewport
   <img src="/wiki-images/1768739771507-yn3w3j.png" alt="Image" width={200} />
2. Hide any non-custom meshes (`H`)
3. Check vertex count (aim for \<60k)
   <img src="/wiki-images/1768739904291-lle6hg.png" alt="Image" width={200} />

If too high:

- Go to the modifiers tab:
  <img src="/wiki-images/1768739963121-r7xh8.png" alt="Image" width={300} />
- Add a **"Decimate" Modifier**
- Use **Collapse Mode**
- Enable **Triangulate**
- Lower the ratio until vertex count is acceptable

If quality degrades too much:

- Try **Planar Mode** with **UV Delimiting** enabled
- You can even stack both collapse and planar decimates if needed

## 1. Edit Your Model to Fit the League Rig

### 1. Import the League Model

Import the League champion model for reference

---

### 2. If the Custom Model Has an Armature

- Select the armature (orange icon not the green)
- <img src="/wiki-images/1774322132344-7oqnj.png" alt="Image" width={300} />
- Enter **Pose Mode**
- Move/Rotate bones to match League proportions

- Once you're done, duplicate the **Armature Modifier**, apply the top one
  <img src="/wiki-images/1774323618655-wzqsmk.png" alt="Image" width={60} />

<img src="/wiki-images/1768740740602-3ws0r86.png" alt="Image" width={200} />

- In Pose Mode, press **Ctrl + A → Apply Pose as Rest Pose**
- Continue editing

---

### 3. Edit Without Using an Armature

- Select the model
- Enter **Edit Mode**
- Press **O** to enable **Proportional Editing**
- Move vertices to reshape the mesh
- Use the mouse wheel to adjust influence

If unwanted areas move:

- Enable **Connected Only** in proportional editing:
  <img src="/wiki-images/1768740885313-m3oqya.png" alt="Image" width={200} />

---

## 2. Combine UVs & Textures

(This part can be annoying, if there are too many materials and you have installed the CATS plugin go to the **Material Atlas** guide page and follow that.)

If the model uses multiple textures:

- Combine them into a single texture kinda like this:
  <img src="/wiki-images/1768741016350-5ad5t.png" alt="Image" width={300} />

### UV Editing

1. Go to the **UV Editing** tab:
   <img src="/wiki-images/1768741066298-t6pn.png" alt="Image" width={200} />
2. Enter **Edit Mode** with the just mesh selected
3. Select a material in the materials tab (make sure you have no other vertices selected before)
4. Replace its image with the combined texture here:
   <img src="/wiki-images/1768741108530-wihx3n.png" alt="Image" width={200} />
5. Click "Select":
   <img src="/wiki-images/1768741185080-h69aki.png" alt="Image" width={200} />
6. The vertices with that material will light up like this:
   <img src="/wiki-images/1768741227212-2agmm.png" alt="Image" width={300} />
7. Scale and move the selceted parts in the UV windom on the left to fit the new texture (change the background image to your new one in the box shown above to make that easier).

Repeat for all materials until all of them use the combined texture and the model looks correct in the viewport.

You can now delete all materials except one.

## 3. Weights & Rigging

### 1. Parenting

If the model already has an armature:

- Select the mesh
- Press **Alt + P → Clear Parent (Keep Transformation)**

Then:

1. Select the mesh
2. Shift-select the League armature
3. Press **Ctrl + P → Armature Deform**

---

### 2. Transferring Weights

Follow this guide to transfer weights from the League model:  
https://www.youtube.com/watch?v=IPQ2QVVIxLU

After transfer:

- Hide the League mesh
- Select the armature
- Enter **Pose Mode**
- Rotate bones to test if the weights transferend correctly. Pay extra attention to the fingers, and any detached parts.

My rule of thumb is you should be able to rotate a bone ~90° without the mesh starting to look weird. It will most likely be really off in some places. To fix it, exit pose mode and go to step 3:

---

### 3. Manual Weight Painting

- Select bones, then the mesh
- Enter **Weight Paint Mode**
- **Alt + Click** a bone to view its influence

Color meanings:

- Red: full weight (the part will exactly follow the bone's movement)
- Yellow/Green: partial
- Blue: no weight (the part is not affected)

Controls:

- **Left Click**: add weight
- **Shift + Left Click**: smooth weight
- **Ctrl + Left Click**: remove weight

Look out for weights where there shouldn't be any. For example, if you see something like this:

## <img src="/wiki-images/1768743573280-9m5v1.png" alt="Image" width={300} />

### 4. Weight Editing via Edit Mode

To remove all weight from a part of the model:

1. Select the mesh
2. Enter **Edit Mode**
3. Select vertices you don't want weights on
4. Open **Vertex Groups**:
   <img src="/wiki-images/1768743909642-ady2ej.png" alt="Image" width={300} />
5. Select the group that shares the name with the bone you don't want weights on
6. Click **Remove**

To assign full weight:

- Select vertices
- Choose vertex group
- Click **Assign**

(Useful for weapons)

---

### 5. Double Check Weights

- Enter **Pose Mode**
- Move the **root bone**

If parts stretch or stay behind like this:

<img src="/wiki-images/1768744036388-2d5f2.png" alt="Image" width={300} />

- They have no weights
- Return to Weight Paint Mode and fix

---

## Final Steps

1. Convert the combined texture to `.tex` using **Quartz** or **LtMAO**
2. Replace the default champion texture
3. Delete the default model in Blender
4. Export skn/skl
5. Replace the original files in the WAD folder
6. Load the mod using your skin manager of choice

If something looks wrong in-game, carefully re-check each step above.
