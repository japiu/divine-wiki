## What You Need

1. **Thumbnail template (.PSD) and font** - [Download from Google Drive](https://drive.google.com/drive/folders/1U211qsFT13Th91ACx6W4GLaEfCta_h1q?usp=sharing)
2. **An image editor** that can open PSD files:
   - [Adobe Photoshop](https://www.adobe.com/products/photoshop.html) (paid)
   - [Photopea](https://www.photopea.com/) (free, works in your browser)
   - [GIMP](https://www.gimp.org/) (free, desktop app)

## Setup

1. Download the PSD template and the font file from the Google Drive link above.
2. Install the font on your system before opening the template (double-click the font file and hit Install).
3. Open the `.psd` template in your editor of choice.
