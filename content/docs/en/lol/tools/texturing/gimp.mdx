GIMP is a free, open-source image editor, the closest thing to a free
Photoshop. For League modding, install the
[Gimp Tex Plugin](/docs/lol/tools/texturing/gimp-tex-plugin) so it can open `.tex`
files directly.

[Download GIMP](https://www.gimp.org/downloads/)

---

## What you'll use it for

- Edit champion splash arts and loading screens.
- Modify or create custom textures.
- Work with `.dds` and `.tex` files (with plugins).
- Make transparent PNGs for UI elements.

GIMP only handles 2D image work. For 3D models, use
[Maya](/docs/lol/tools/3d-modeling/maya) or [Blender](/docs/lol/tools/3d-modeling/blender).
