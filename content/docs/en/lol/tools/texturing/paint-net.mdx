Paint.NET is a free, lightweight image editor for Windows. The simplest
option for League texture work if Photoshop and GIMP feel like overkill.
For League modding, install the
[Paint.NET Tex Plugin](/docs/lol/tools/texturing/paint-net-tex-plugin) so it
can open `.tex` files directly.

[Download Paint.NET](https://www.getpaint.net/download.html)

---

## What you'll use it for

- Edit champion splash arts and loading screens.
- Modify or create custom textures.
- Work with `.tex` files (with the plugin).
- Make transparent PNGs for UI elements.

Paint.NET only handles 2D image work. For 3D models, use
[Maya](/docs/lol/tools/3d-modeling/maya) or [Blender](/docs/lol/tools/3d-modeling/blender).
