# LoL Blender Plugin — Quick Guide

**Blender addon for importing/exporting League of Legends models, animations, and static objects.**


[LoL Blender Plugin | Download from GitHub](https://github.com/RitoShark/Aventurine-League-Tools)

---

## Key Features

| Feature | Description |
|---------|-------------|
| **SKL + SKN** | Import/Export character models with skeletons |
| **ANM** | Import/Export animations |
| **SCB / SCO** | Import/Export static objects |
| **Bone Limit** | Limit to max 4 influences per vertex |
| **UV Tools** | UV corner snapping utilities |
| **Drag & Drop** | Easy file handling |

---

## Requirements

| Requirement | Details |
|-------------|---------|
| Blender | **4.0+** (Windows only) |
| FPS Setting | **30 FPS** recommended for animations |

---

## Installation

1. Download [`Aventurine.zip`](https://github.com/RitoShark/Aventurine-League-Tools/releases) — **Do not unzip!**
2. Open Blender → `Edit` → `Preferences` → `Add-ons` → `Install…`
3. Select the `.zip` file → Click **Install Add-on**
4. Enable **Aventurine: League Tools**

> ✅ **Verify:** Import/Export options should appear in the **File menu**, and **Aventurine LoL** tab in sidebar (`N` key)

---

### Usage Guide

<Tabs items={["Import", "Export"]}>
<Tab value="Import">

### Importing Models

1. Go to `File → Import → LoL SKN+SKL`
2. Select your `.skn` file (ensure `.skl` is in the same folder)
3. Adjust scale (default: `0.01`) → Click **Import**

### Importing Animations

1. First import your model (SKL+SKN)
2. Go to `File → Import → LoL ANM Animation`
3. Select your `.anm` file

</Tab>
<Tab value="Export">

### Exporting Models

1. Select your **mesh + armature** in the viewport
2. Go to `File → Export → LoL SKN+SKL`
3. Use **Limit to 4 Influences** if needed before export

### Exporting Animations

1. Select your **armature** with the animation
2. Go to `File → Export → LoL ANM Animation`
3. Choose output location

</Tab>
</Tabs>

## LoL Tools Sidebar

Press `N` to open the sidebar, then find the **LoL Tools** tab:

| Tool | Function |
|------|----------|
| Import/Export | Quick access shortcuts |
| Limit to 4 Influences | Fix bone weights for export |
| UV Corners | Snap UV vertices |

---

## 💡 Quick Tips

- Default import scale = **0.01** (League units are large)
- Max **4 bone influences** per vertex is required for export
- Always set FPS = **30** for animations
- Keep `.skl` and `.skn` files in the **same folder**
