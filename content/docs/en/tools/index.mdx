If you don't know where to start, grab [**Flint**](/docs/tools/flint)
(does almost everything), [**Jade**](/docs/tools/jade) for BIN editing,
and [**LtMAO**](/docs/tools/ltmao) as a backup for the odd job.
Everything else below is sorted by the part of the workflow it fits.

---

## Extract & browse game files

<ToolCard name="Obsidian" href="/docs/tools/obsidian">
  The classic tool for digging through League's game files.
</ToolCard>
<ToolCard name="Flint — WAD Explorer" href="/docs/tools/flint">
  Built into Flint. Browse the whole game like a folder tree, no need to extract
  first.
</ToolCard>

---

## BIN editing

For game logic, links, materials, and particle definitions.

<ToolCard name="Jade" href="/docs/tools/jade" badge="Recommended">
  Edit BIN files with a real GUI. Syntax coloring, particle preview, auto hash
  lookups.
</ToolCard>
<ToolCard name="RitoBin" href="/docs/tools/ritobin" badge="CLI">
  Command-line BIN converter. Older, but still solid for batch jobs and scripts.
</ToolCard>
<ToolCard name="Hacksaw" href="/docs/tools/Hacksaw">
  Edits VFX BIN files. Good for recoloring particles.
</ToolCard>

---

## All-in-one apps

If you'd rather not juggle five tools, these do most of it from one window.

<ToolCard name="Flint" href="/docs/tools/flint">
  Extract assets, preview models and textures, edit BIN files, and pack the mod
  — all in one app.
</ToolCard>
<ToolCard name="Quartz" href="/docs/tools/tex-thumbnail-provider">
  Edits BINs, textures, and audio. Pairs with Jade and Flint.
</ToolCard>
<ToolCard name="LtMAO" href="/docs/tools/ltmao">
  Toolpack covering extraction, BIN conversion, audio conversion, and mod
  packing. Good fallback when something else won't work.
</ToolCard>

## Texture tools

<Tabs items={["Editors (free)", "Editors (paid)", "Editor plugins", "Converters"]}>
<Tab value="Editors (free)">

<ToolCard name="GIMP" href="https://www.gimp.org/downloads/">
  Free image editor. Handles `.dds` textures with the right plugin.
</ToolCard>
<ToolCard name="Paint.NET" href="https://www.getpaint.net/">
  Lightweight Windows image editor. Good for quick palette work.
</ToolCard>

</Tab>
<Tab value="Editors (paid)">

<ToolCard name="Photoshop" href="/docs/tools/photoshop">
  The industry standard. Best for serious texture work.
</ToolCard>

</Tab>
<Tab value="Editor plugins">

<ToolCard name="Gimp Tex Plugin" href="/docs/tools/gimp-tex-plugin">
  Lets GIMP open League's `.tex` files directly.
</ToolCard>
<ToolCard name="Paint.NET Tex Plugin" href="/docs/tools/paint-net-tex-plugin">
  Lets Paint.NET open League's `.tex` files directly.
</ToolCard>
<ToolCard name="Photoshop Tex Plugin" href="/docs/tools/photoshop-tex-plugin">
  Lets Photoshop open League's `.tex` files directly.
</ToolCard>

</Tab>
<Tab value="Converters">

<ToolCard
  name="tex2dds"
  href="https://github.com/Morilli/Ritoddstex/releases"

>

</ToolCard>
<ToolCard name="TexThumbnailProvider" href="/docs/tools/tex-thumbnail-provider">

</Tab>

## Audio

<Tabs items={["LoL audio", "Editors", "Advanced"]}>
<Tab value="LoL audio">

<ToolCard name="LtMAO" href="/docs/tools/ltmao" badge="Recommended">
  Best default for converting `.wem` to `.wav` and back.
</ToolCard>
<ToolCard name="Quartz" href="/docs/tools/tex-thumbnail-provider">
  Has a built-in `.wpk` editor for League soundbanks.
</ToolCard>
<ToolCard name="Foobar2000" href="https://www.foobar2000.org/">
  Music player that can also convert `.wem` to `.wav`. Skip unless you already
  use it.
</ToolCard>

</Tab>
<Tab value="Editors">

<ToolCard name="Audacity" href="https://www.audacityteam.org/">
  Free `.wav` editor. Cut, layer, and export — the basics.
</ToolCard>

</Tab>
<Tab value="Advanced">

<ToolCard
  name="Wwise 2019"
  href="https://www.audiokinetic.com/en/download/"
  badge="Advanced"
>
  Pro audio middleware. Only needed if you're rebuilding full soundbanks.
</ToolCard>

</Tab>
</Tabs>

## Mod packaging & install

<ToolCard name="CS-LoL Manager" href="/docs/tools/cs-lol-manager">
  Installs mods and packs them as `.fantome` files.
</ToolCard>
<ToolCard name="LTK Manager" href="https://github.com/LeagueToolkit/Hematite">
  Mod manager from the LeagueToolkit team. Flint can sync mods to it directly.
</ToolCard>
<ToolCard name="LtMAO" href="/docs/tools/ltmao">
  Also packs mods. See **All-in-one apps** if you want one tool that does
  everything.
</ToolCard>
<ToolCard name="League Mod Repather" href="/docs/tools/league-mod-repather">
  Re-paths assets so two mods touching the same champion don't conflict.
</ToolCard>
