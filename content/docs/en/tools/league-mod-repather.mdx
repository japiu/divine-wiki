# League Mod Repather

A standalone tool to automatically fix League of Legends mods after game updates.

When Riot updates League of Legends, old mods often break. This tool automatically updates your mods to work with the latest game version by repathing them.

https://github.com/RitoShark/League-Mod-Repather/releases/tag/v2.0.2

---

## How to Use

### First Time Setup

1. Download and run `LeagueModRepather.exe`
2. Click **Download** to get the hash files
3. Wait for download to complete (~1 minute)

---

### Step 1: Select Files

**Champions Folder:**
- Click **Browse** next to "Champions folder"
- Navigate to: `C:\Riot Games\League of Legends\Game\DATA\FINAL\Champions`
- Click **Select Folder**

**Your Mod:**
- **Option A:** Select a `.fantome` file
- **Option B:** Select a mod folder (for mod creators only)

**Custom Prefix** (Optional):
- Leave blank for automatic (recommended)
- Or enter your own prefix

Click **Next**

---

### Step 2: Processing

The tool will automatically:
- Detect the champion
- Extract and process files
- Convert textures

Processing takes 2-5 minutes. Wait for: `Overlay complete`

Click **Next**

---

### Step 3: Select Main Skin

Choose from the dropdown (usually **Skin0**) and click **Next**

---

### Step 4: Repathing

The tool will fix all file paths and create your new mod.

When complete: `DONE! Created {name}_repathed.fantome`

---

### Output Location

Your repathed mod is saved to:
```
C:\Users\<YourName>\Documents\FantomeRepathTool\
```

The `.fantome` file will also be in the same location as the original.

---

## Installation

1. Open your mod manager (Fantome, LCS Manager, etc.)
2. Import the `_repathed.fantome` file
3. Enable and play

---

## Troubleshooting

**Mod still doesn't work:**
- Ensure you selected Skin0 (or the correct skin number)
- Click **Update** to refresh hash files
- Try repathing again

**Tool stuck on "Converting TEX to DDS":**
- This is normal for large mods
- Wait 2-5 minutes

**Hash file location:**
```
C:\Users\<YourName>\AppData\Roaming\FrogTools\hashes\
```
