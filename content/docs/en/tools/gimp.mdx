## What is GIMP?

**GIMP** (GNU Image Manipulation Program) is a free and open-source raster graphics editor. It's a powerful alternative to Photoshop for:

- **Photo Editing** – Retouch, enhance, and manipulate images
- **Texture Creation** – Create and edit textures for game assets
- **Image Composition** – Combine multiple images with layers and masks
- **Digital Art** – Brushes, gradients, and painting tools
- **File Conversion** – Export to various formats (PNG, DDS, TGA, etc.)

### Why Use GIMP for LoL Modding?

GIMP is useful for League modding when you need to:

- Edit champion **splash arts** and **loading screens**
- Modify or create custom **textures**
- Work with **DDS files** (with the DDS plugin)
- Create **transparent PNGs** for UI elements
- Batch process images

> **Note:** For 3D model editing, you'll need [Maya](/docs/maya/lol-maya-plugin) or [Blender](/docs/blender). GIMP is for 2D image work only.
