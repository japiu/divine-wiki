RitoBin is a command-line tool by **Moonshadow** that converts League's
`.bin` files into readable `.py` code and back. Older than Jade, but
still solid for batch jobs and scripts.

https://github.com/moonshadow565/ritobin/releases/tag/2025-10-05-e686d9e

---

## Installation & Setup

1. Download the latest version from GitHub.
2. Create a dedicated folder for **RitoBin** and your `.bin` files.
3. Extract the contents of the download into that folder.

> ⚠️ **Important:** Make sure the **hashes** folder stays in the same directory as both `ritobin_cli.exe` and `ritobin_gui.exe`. If the hashes are missing or misplaced, conversions won't work properly.

### Converting Files

<Tabs items={[".bin → .py", ".py → .bin"]}>
<Tab value=".bin → .py">

### Converting `.bin` to `.py`

1. Make a `ritobin_cli.exe` shortcut to your desktop.
2. Drag and drop the `.bin` file onto `ritobin_cli.exe`.

The tool will generate a `.py` file — this is a human-readable version of the `.bin`, which you can now edit.

You can open the `.py` file in any text editor, such as **VS Code**, **Notepad++**, or any IDE of your choice.

</Tab>
<Tab value=".py → .bin">

### Converting `.py` Back to `.bin`

Once you've made your changes to the `.py` file and you're ready to test it in-game:

1. Drag and drop the edited `.py` file onto `ritobin_cli.exe`.

The tool will convert it back into a working `.bin` file.

</Tab>
</Tabs>

## 💡 Tip: Quick Convert with Default Program

You can set `ritobin_cli.exe` as the default program for `.py` files. This way, you can convert back to `.bin` just by double-clicking the `.py` file — no need to drag and drop every time.

| Step | Action                                        |
| ---- | --------------------------------------------- |
| 1    | Right-click the `.py` file                    |
| 2    | Select **Open with** → **Choose another app** |
| 3    | Select `ritobin_cli.exe`                      |
| 4    | Check **Always use this app**                 |
