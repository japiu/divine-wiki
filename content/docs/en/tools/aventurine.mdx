Aventurine is a **Blender** addon by **Bud and Frog** (Divine team)
that adds native support for League's file formats. It's the reason
Blender is now a real option for League modding instead of a workaround.

[Download the latest release](https://github.com/RitoShark/Aventurine-League-Tools/releases)

---

## What it does

- **Import and export** `.skn` meshes, `.skl` skeletons, and `.anm`
  animations — no external converters.
- **Physics simulation** for hair, cloth, and accessories. Built on
  Wiggle 2.
- **Animation retargeting** — move animations between different rigs.
- **Auto-updater** — checks for new versions inside Blender.

---

## Requirements

- **Blender 4.0** or newer.

---

## Install

1. Download the latest `.zip` from the
   [releases page](https://github.com/RitoShark/Aventurine-League-Tools/releases).
2. Open Blender.
3. Go to **Edit → Preferences → Add-ons**.
4. Click **Install** and pick the downloaded `.zip`.
5. Search for "Aventurine" in the addon list and tick the checkbox.

The addon is ready to use.

---

## How to find it inside Blender

1. Open the **3D Viewport**.
2. Press **N** to open the side panel.
3. Click the **Aventurine LoL** tab.

All import / export buttons live in this panel.

---

## Import and export

- Use the buttons in the **Aventurine LoL** panel to import `.skn`,
  `.skl`, and `.anm` files.
- Before exporting, double-check that the right armature or mesh is
  selected.

---

## Physics simulation

The physics tools are off by default to keep the panel clean.

1. Go to **Edit → Preferences → Add-ons → Aventurine**.
2. Tick **Animation Tools**.
3. A new **Animation Tools** tab appears in the side panel.

Use it to add secondary motion to hair, cloth, and accessories without
hand-keyframing.

---

## Animation retargeting

The **LoL Retarget** option lets you move an animation from one rig to
another. Useful for:

- Adapting an animation to a custom rig.
- Sharing animations across champions.
- Testing animations on different character models.

Enable it in the addon preferences first.

---

## Updating

1. Go to **Edit → Preferences → Add-ons**.
2. Find **Aventurine** in the list.
3. Expand the addon details.
4. Click **Check for Updates**.
5. If an update is available, click to install.

---

## When to use Aventurine vs Maya

- **Use Aventurine + Blender** if you don't want to pay for Maya, or
  you already know Blender.
- **Use [Maya + lol_maya](/docs/tools/maya)** if you want the most
  established workflow, the largest tutorial library, and a polished
  rigging pipeline.

Both work. Pick what you'll actually use.
