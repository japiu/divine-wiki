VS Code is a free, lightweight code editor. You'll use it to edit the
`.py` files RitoBin produces, or to write Python addons for Aventurine
and lol_maya.

---

## Download

[**Download VS Code** | Official download page for Windows, Mac, and Linux](https://code.visualstudio.com/)

---

## Why Use It?

- ✅ Syntax highlighting for `.py` files
- ✅ Easy to read and modify League's particle scripts
- ✅ Works great with RitoBin output
- ✅ Tons of extensions, themes, and tools

---

## 💡 Tips

> **Use the Python extension** for better syntax support
>
> Press `Ctrl + /` to quickly comment/uncomment lines
>
> Drag and drop `.py` files directly into the editor to start editing
