Tex Thumbnail Provider by **GuiSai** lets you see `.tex` files as
thumbnails in Windows Explorer. No more guessing what's inside.

[Tex Thumbnail Provider | Preview .tex files in Windows Explorer](https://github.com/RitoShark/TexThumbnailProvider)

---

## Installation

1. Download `installer.rar` from the GitHub page.
2. Extract and run the installer.

> ✅ The installer automatically registers the necessary DLLs. No manual steps are usually required.

---

## ⚠️ Important

| Rule                                               | Why                       |
| -------------------------------------------------- | ------------------------- |
| Do **not** move the `.dll` file after installation | Breaks thumbnail previews |
| Do **not** rename the `.dll` file                  | Windows won't find it     |

<Tabs items={["Quick Fixes", "Manual Fix"]}>
<Tab value="Quick Fixes">

### Thumbnails Not Showing?

1. **Verify installation path** — DLL should be where you installed it
2. **Check permissions** — Must be installed with Administrator rights
3. **Restart Explorer** — Sometimes Windows needs a refresh

</Tab>
<Tab value="Manual Fix">

### Re-register the DLL Manually

Open **Command Prompt as Administrator** and run:

```cmd
regsvr32 "C:\TexThumbnailProvider\texthumbnailprovider_by_guisai_v2.dll" /u
regsvr32 "C:\TexThumbnailProvider\texthumbnailprovider_by_guisai_v2.dll"
```
