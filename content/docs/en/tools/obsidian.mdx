Obsidian is the classic tool for digging through League's game files.
Built by **Crauzer**. You'll use it to browse, preview, and extract
files from League's `.wad.client` archives.

https://github.com/Crauzer/Obsidian/releases/tag/5.1.8

---

## Download & Installation

### Download Obsidian

> ⚠️ **Avoid pre-release versions** unless you know what you're doing — they may be unstable.

### How to Install

1. Download the `.zip` file from GitHub.
2. Extract the archive into a folder of your choice.
3. Launch `Obsidian.exe`.

### First-Time Setup

When you open Obsidian for the first time, it will ask you to locate your **League of Legends Game folder**.

**Default path:**

C:\Riot Games\League of Legends\Game

If you installed League elsewhere, browse to your custom install location. Once selected, **restart Obsidian** — and you're ready to go.

## How to Use Obsidian

### Browsing Game Files

After launching Obsidian and letting it load, you'll see the internal folder structure of League's assets.

1. Open the `DATA` folder (click the arrow to expand).
2. Then open the `FINAL` folder — this is where the real content lives.

### Inside you'll find folders like

| Folder        | Contains                               |
| ------------- | -------------------------------------- |
| **Champions** | Champion models, VFX, animations, etc. |
| **Maps**      | Summoner's Rift, Howling Abyss, etc.   |
| **UI**        | HUD, Icons, Ability panels, and more   |

> 💡 Most assets can be previewed directly in Obsidian (like `.dds` and `.tex` textures)!

---

### Understanding `.wad.client` Files

Every champion has at least two types of `.wad.client` files:

| File Type                 | Contains                             |
| ------------------------- | ------------------------------------ |
| `Aatrox.wad.client`       | Visual files (models, textures, VFX) |
| `Aatrox.en_US.wad.client` | Sound files (Voiceover)              |

> 💡 The localization file will differ depending on your language (e.g. `fr_FR`, `de_DE`, etc.).

---

### Extracting Files

1. Select the files or folders you want to extract. **ALWAYS EXTRACT THE WHOLE WAD**
2. Go to the top menu: **File → Extract Selected**
3. Choose a folder where you want the files to be saved.

> ⚠️ **Important:** Avoid clicking "Extract All" unless you intend to dump the entire League file system (this includes many GBs of data). Don't extract only parts of the `.wad.client`
